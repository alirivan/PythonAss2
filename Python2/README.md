# Scrapping data from web-sites

Here information is scraped from coinmarketcap.com.


<h1>Installation</h1>

```
If you use Python 3, then use these codes

pip3 install requests
pip3 install  bs4

-----------------------------------------

If you use Python 2 use following

pip install requests
pip install  bs4

```

```
import requests

```
